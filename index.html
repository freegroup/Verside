( What is Verside )

Verside is a very popular and mature Open Source HTML5 and PHP application which allows you to easily create a highly customizable database front-end/application without coding.

You can use it to create in a few minutes a very basic CRUD (create, read, update, delete) database front-end or to develop a more sophisticated database application. No programming skills are needed to use it; if however you are a PHP programmer with Verside you can save tens of hours of work.

With Verside (differently from many competitors) you don't produce a PHP script, which would become outdated when you modify the schema of your database, but instead you directly use its abstraction layer that can be easily updated every time you modify your database's schema.

Verside uses thes Doctrine database abstraction libraries in order to support as many DBMSs as possible, at the moment it has been tested on MySQL and SQLite.

( Documentation )

Requirements

  • + PHP
    • - version >= 5.3 (with SQLite support)
    • - PDO
  • + Database
    • - MySQL version >= 3.23.x
    • - SQLiteversion >= 3 (it could work with version 2 too, not tested)
A pre-existent database with AT LEAST ONE table to manage, with Verside you can't create databases and tables,

Installation

  • + Copy the installer.php in a directory under your Web server
  • + Run the file http://your_host/your_dir/installer.php to install Verside
You can now create your first form to insert, search, update, delete the records of the tables contained in the database. You should have set the primary key for each table if you want to delete and update records, otherwise Verside won't show you the corresponding buttons.

Security

After the program is installed it is a good practice to remove the file installer.php and all the ./installer/* files since they could be used by malicious users in order to change or even re-install your Verside installation.

Authentication

An authorization model is available to prevent unauthorized accesses to the application. You have enterd the usename and password during the installation process.
